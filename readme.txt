Shows WordPress admin styles to help WordPress developers create plugins.

== Description ==

This plugin shows built-in WordPress admin styles to help WordPress developers create plugins with user interfaces (UI) that match existing admin styles.

== Frequently Asked Questions ==

= Where do I find the Admin Style Book? =

From your WordPress dashboard, go to Tools > Admin Style Book.
